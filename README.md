# 我的私人食谱

一个轻量级的个人菜谱记录网站，支持多人共同编辑，数据实时同步。

## 功能

- 记录菜谱：菜名、食材、做法步骤、图片、视频链接、成本、耗时
- 多人实时同步（Firebase Firestore）
- AI 辅助填写：右下角悬浮助手，支持问答记录或粘贴字幕自动整理
- B站链接识别：自动抓取视频字幕辅助整理菜谱
- 今天吃什么：按场合/耗时筛选随机推荐
- 购物清单：多道菜食材一键汇总
- 导入/导出：支持 JSON 备份

## 在线访问

**https://pokkan39.github.io/my-recipes/**

## 本地运行

直接用浏览器打开 `index.html` 即可，无需安装任何依赖。

## AI 助手配置

点击右下角 ✦ 按钮 → 齿轮图标 → 填写：

- **API Key**：你的 OpenAI 或兼容接口 Key
- **接口地址**：默认 `https://api.openai.com/v1`，使用中转站填中转地址
- **模型**：默认 `gpt-4o`

API Key 仅存储在本地浏览器，不会上传到任何服务器。

## 技术栈

- 纯原生 HTML / CSS / JavaScript，无框架
- Firebase Firestore（实时数据库）
- OpenAI 兼容 API（AI 助手）
