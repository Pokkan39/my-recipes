# 我的食谱

这是一个正在内测中的朋友共享菜谱库。

我们想做的不是一个冷冰冰的表格工具，而是一个大家都愿意顺手打开、顺手补充的美食小站。家里常做的菜、朋友推荐的菜、视频里刷到的做法，都可以慢慢收进来。以后不知道今天吃什么时，就从这里挑；看到一个做菜视频，也可以让 AI 帮忙整理成真正能照着做的菜谱。

在线访问：

**https://pokkan39.github.io/my-recipes/**

## 为什么做这个

很多好吃的做法其实都散落在聊天记录、收藏夹、短视频和脑子里。时间一长，要么找不到，要么只记得“好像看过”。

这个项目想解决一个很简单的问题：

> 把大家觉得好吃、实用、想复做的菜，整理到一个共同的地方。

现在还只是内测版，但已经可以多人一起填写。你可以把自己会做的菜补进去，也可以把看到的视频链接或字幕交给 AI，让它先整理一版菜谱草稿。

## 现在能做什么

- 记录菜谱：菜名、来源、版本、介绍、食材、步骤、成本、耗时、适合时间
- 多人共同编辑：大家打开同一个网站，看到同一份菜谱数据
- AI 辅助填写：右下角 AI 助手可以问答式整理菜谱
- 视频辅助整理：粘贴 B站链接后，系统会尝试获取标题、简介和字幕
- 一键填入表单：AI 整理出菜谱后，可以直接填入编辑表单
- 今天吃什么：从已有菜谱里随机推荐
- 购物清单：按多道菜汇总食材
- 导入和导出：支持 JSON 备份

## 怎么参与填写

如果你也想一起补菜谱，可以直接打开网站：

**https://pokkan39.github.io/my-recipes/**

然后可以做这些事：

1. 点“新增菜谱”，把你会做的菜录进去
2. 点右下角 AI 按钮，让 AI 帮你整理菜谱
3. 看到 B站做菜视频，可以把链接粘给 AI 试试
4. 如果 AI 没抓到字幕，可以把视频文案或字幕复制进去
5. 保存前可以自己检查一遍，避免 AI 写错用量或步骤

现在是内测阶段，所有人都能编辑。看到问题也可以直接改，或者把建议告诉我们。

## AI 助手怎么用

点击右下角的 AI 按钮，第一次使用需要填写：

- API Key：你的 OpenAI 或兼容接口 Key
- 接口地址：默认是 `https://api.openai.com/v1`，使用中转站就填中转地址
- 模型：默认 `gpt-4o`

API Key 只会保存在你自己的浏览器里，不会上传到网站数据库。

AI 适合做这些事：

- 把一段做菜文字整理成菜谱
- 根据视频字幕提取食材和步骤
- 帮你补充小白提示
- 帮你估算耗时和成本
- 根据已有食材推荐一道菜

需要注意：AI 不能真正“看视频画面”。如果视频没有公开字幕，AI 只能根据标题、简介、你粘贴的文案或你补充的信息来整理。

## 当前内测限制

这个版本还在打磨中，有几个地方需要大家理解：

- 当前所有人都可以编辑，暂时没有登录和权限控制
- 如果有人误删内容，需要通过备份恢复
- 视频字幕抓取不保证每次成功，取决于视频是否有公开字幕和接口是否可访问
- AI 生成内容需要人工检查，尤其是用量、火候、时间和安全相关步骤
